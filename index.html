Introducción

Este repositorio recoge buenas prácticas y plantillas para proyectos editoriales. Usa la barra lateral para navegar entre las secciones. Las páginas específicas están preparadas como plantillas debajo de este archivo (comentadas) para que puedas separarlas si lo deseas.

Sistemas de columnas, módulos y rejillas para coherencia visual.
